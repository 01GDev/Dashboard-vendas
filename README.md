

---

Análise de Vendas

Projeto voltado para explorar dados de vendas e entender padrões de faturamento e comportamento dos clientes.


---

Objetivo

Identificar quais fatores impactam as vendas, como período, perfil de cliente e categorias de produtos.


---

O que foi feito

Organização e limpeza dos dados

Criação de tabelas dinâmicas

Análise de vendas por período

Análise por faixa etária

Comparação entre categorias

Criação de gráficos para visualização



---

Ferramentas

Excel

Tabelas dinâmicas

Gráficos



---

Principais insights

Desempenho ao longo do tempo

Identificação do melhor e pior mês de vendas


Perfil dos clientes

Clientes de 43 anos representam 12,55% das compras


Categorias

Categoria com maior volume de vendas identificada



---

Conclusão

Os dados mostram que existem oportunidades de focar em perfis específicos de clientes e nas categorias mais vendidas para melhorar os resultados.
